Myriagon
========

Python 3.4+ only.

Installation
------------

macOS Installation
~~~~~~~~~~~~~~~~~~

.. code-block:: sh

   python3 -m pip install https://github.com/Tinche/cattrs/archive/master.zip
   python3 -m pip install myriagon

Linux Installation
~~~~~~~~~~~~~~~~~~

.. code-block:: sh

   apt-get install python3-gi # Or similar for your distro
   python3 -m pip install https://github.com/Tinche/cattrs/archive/master.zip
   python3 -m pip install myriagon

Running
-------

``python3 -m myriagon``
